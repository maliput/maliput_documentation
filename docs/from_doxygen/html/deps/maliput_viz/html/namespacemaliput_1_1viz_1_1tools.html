MaliputVizConfig LoadYamlConfigFile ( const std::string &  file_path)

Parses a YAML file to be used by the maliput_viz application.

The YAML file must have the following structure:

maliput_viz:"
maliput_backend: <backend_name> "
parameters: "
<key_1>: <value_1> "
<key_2>: <value_2> "
// ...
<key_N>: <value_N> "
