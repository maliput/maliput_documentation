◆ BuildGraph()

Graph BuildGraph ( const api::RoadGeometry *  rg)

Builds a Graph from rg.

Matches the api::Segments in rg to each Node, and bundles api::BranchPoints at the extents of an api::Segment whose api::Lanes on one side connect to another set of api::Lanes belonging to another api::Segment.

Parameters
rgapi::RoadGeometry to build a Graph from. It must not be nullptr.
Returns
A Graph.
Exceptions
maliput::common::assertion_errorWhen rg is nullptr.

const Edge &  ref_edge,

Determines the api::Lane::End::Which end of an api::Lane in ref_edge that connects to an api::Lane in target_edge.

When there is no direct connection between ref_edge and target_edge, std::nullopt is returned. Note: this method uses the graph connectivity, thus there might be no real connection between ref_edge and target_end underlying api::Segments by means of api::BranchPoints, thus returning std::nullopt. When ref_edge.segment and target_edge.segment are equal, this function returns std::nullopt.

Parameters
graphThe Graph containing ref_edge and target_edge.
ref_edgeThe reference Edge.
target_edgeThe target Edge.
Returns
An optional with the api::LaneEnd::Which end of an api::Lane in ref_edge that connects with another api::Lane in target_edge.
