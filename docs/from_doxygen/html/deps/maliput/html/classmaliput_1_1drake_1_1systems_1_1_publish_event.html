Detailed Description

template<typename T>
class maliput::drake::systems::PublishEvent< T >

This class represents a publish event.

It has an optional callback function (two forms are supported) to do custom handling of this event.

See also
System::Publish for more details.
LeafSystem for more convenient interfaces to publish events via the Declare*PublishEvent() methods.

#include <include/maliput/drake/systems/framework/event.h>

Public Types

Publish Callbacks

Two callback signatures are supported.

See Event Callbacks for a complete discussion. Both callback types receive a const reference to the context and this publish event. They differ in whether a const reference to the owning system is also provided.
