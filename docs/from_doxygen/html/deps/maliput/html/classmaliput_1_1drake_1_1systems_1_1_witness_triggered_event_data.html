WitnessTriggeredEventData< T > Class Template Reference

Detailed Description

template<class T>
class maliput::drake::systems::WitnessTriggeredEventData< T >

Class for storing data from a witness function triggering to be passed to event handlers.

A witness function isolates the time to a (typically small) window during which the witness function crosses zero. The time and state at both sides of this window are passed to the event handler so that the system can precisely determine the reason that the witness function triggered.
