Signal Struct Reference

Detailed Description

Holds the values of a XODR Signal Offset.

For example, a XML node describing a XODR's signal offset:

<OpenDRIVE>
...
<signals>
<signal s="3981.4158159146"
t="-14.0503"
id="5000162"
name="Vorschriftzeichen"
dynamic="no"
orientation="+"
zOffset="3.8835"
country="DE"
countryRevision="2017"
type="274"
subtype="100"
value="100"
unit="km/h"
height="0.77"
width="0.77"
hOffset="5.7595865"
length="1.50"
pitch="0.77"
roll="0.77"
text="custom signal text">
</signal>
</signals>
...
</OpenDRIVE>
