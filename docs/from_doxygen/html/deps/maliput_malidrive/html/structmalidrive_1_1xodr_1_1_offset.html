Offset Struct Reference

Detailed Description

Holds the values of a XODR description's offset.

For example, a XML node describing a XODR's offset:

<OpenDRIVE>
<header>
<offset x="1" y="2" z="0" hdg="1.57">
</offset>
</header>
...
...
</OpenDRIVE>
