PiecewiseFunction Class Reference

Detailed Description

Describes a Function defined in pieces.

Queries accept p ∈ [p0, p1] with a linear tolerance.

Constructs PiecewiseFunction from a collection of Functions.

Functions are expected to be C1 continuous and C1 continuity is evaluated at the extents. C0 continuity (position) is checked using linear_tolerance. C1 continuity (tangent/derivative) is checked by comparing the heading angles (computed as atan(f'(p))) against angular_tolerance. Constraints can be disabled with continuity_check.
