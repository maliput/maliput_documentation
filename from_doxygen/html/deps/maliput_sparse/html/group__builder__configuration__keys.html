staticconstexpr

Translation from maliput to maliput_osm inertial frame.

The format of the 3-dimensional vector that is expected to be passed should be {X, Y, Z}. Same format as maliput::math::Vector3 is serialized.

  • Default: "{0., 0., 0.}"
