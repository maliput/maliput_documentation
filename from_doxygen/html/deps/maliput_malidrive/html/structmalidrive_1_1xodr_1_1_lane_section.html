LaneSection Struct Reference

Detailed Description

Holds the values of a XODR LaneSection.

For example, a XML node describing a XODR's lane section:

<OpenDRIVE>
...
<road>
<lanes>
<laneSection s="0.0000000000000000e+0" singleSide="true">
<left>
<lane>
...
</lane>
</left>
<center>
<lane>
...
</lane>
</center>
<right>
<lane>
...
</lane>
</right>
</laneSection>
</lanes>
<road>
...
</OpenDRIVE>
