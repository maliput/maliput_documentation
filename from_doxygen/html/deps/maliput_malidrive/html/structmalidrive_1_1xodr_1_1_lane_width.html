LaneWidth Struct Reference

Detailed Description

Holds the values of a XODR Lane Width.

For example, a XML node describing a XODR's lane width:

<OpenDRIVE>
...
<road>
<lanes>
<laneSection>
<right>
<lane>
<width sOffset="0.0" a="3.5" b="0.0" c="0.0" d="0.0"/>
</lane>
</right>
</laneSection>
</lanes>
<road>
...
</OpenDRIVE>
