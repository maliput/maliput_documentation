Number of trials the Builder will run with different tolerances.

Using default linear tolerance value it is expected that the builder tries with tolerances in a range between 0.05 and 0.49. (kLinearTolerance and kLinearTolerance * kIncreasingToleranceStep ^ kMaxToleranceSelectionRounds )
