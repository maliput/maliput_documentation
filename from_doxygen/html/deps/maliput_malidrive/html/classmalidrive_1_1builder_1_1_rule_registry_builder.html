Builds a maliput::api::rules::RuleRegistry.

The maliput::api::rules::RuleRegistry is built in two steps: 1- Loads maliput::api::rules::RuleRegistry with rule types obtained from the YAML file if it is provided.

2- Programatically loads the following list of rule types:

  • maliput::api::rules::DiscreteValueRule types:
    • See also
      malidrive::builder::rules::VehicleExclusiveRuleTypeId().
    • See also
      malidrive::builder::rules::VehicleUsageRuleTypeId().
    • See also
      maliput::DirectionUsageRuleTypeId().
  • maliput::api::rules::RangeValueRule types:
    • See also
      maliput::SpeedLimitRuleTypeId().
