OpenRangeValidator Class Reference

Detailed Description

Functor to validate if a number is within [min, max] and considering assuming a tolerance.

This tolerance extends the range to be [min - tolerance, max + tolerance]. The functor not only validates that number is within the range, but also adapts it to be in the interval (min, max) where the difference between the close and open range is epsilon. This functor is motivated to wrap MALIDRIVE_DEMAND() and MALIDRIVE_IS_IN_RANGE() calls plus a std::clamp() to interface with ODRM.
